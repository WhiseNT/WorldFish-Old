# WorldFish - 世界观推演引擎

## 项目概述

**WorldFish** 是一个基于 MiroFish 分叉的世界观模拟推演引擎，专为创作者设计。它允许用户输入世界观设定、年代大事记、人物信息、世界走向等数据，然后进行深度的世界观推演。

### 核心功能

- **世界观管理**：支持输入和管理世界观设定、年代大事记、人物信息、世界走向、国家实体、世界地图等信息
- **AI 信息提炼**：通过 AI 将综合性世界观文件自动提炼为结构化数据
- **双模式推演**：
  - **未知推演**：对当前世界观信息未列出的年代进行推演
  - **已知推演**：对已列出的年代进行推演，细化世界观
- **项目持久化**：保存推演项目并长期使用，支持持续迭代
- **世界线管理**：可以增加变数来修改推演世界线，支持时间线回溯和重新推演

## 技术架构

### 前端
- **框架**：Vue 3 + Vite
- **状态管理**：Vuex
- **路由**：Vue Router
- **国际化**：Vue I18n
- **可视化**：D3.js (用于地图和关系图谱)
- **UI 组件**：自定义组件

### 后端
- **框架**：Flask
- **语言**：Python 3.11+
- **AI 集成**：OpenAI SDK
- **记忆管理**：Zep Cloud
- **模拟引擎**：基于 OASIS 框架定制

## 工作流程

1. **世界观构建**：
   - 手动输入世界观设定
   - 上传综合性世界观文件，通过 AI 自动提炼
   - 管理人物、国家、事件等实体

2. **推演配置**：
   - 选择推演模式（未知/已知）
   - 设置推演参数和时间范围
   - 配置 AI 模型和推演深度

3. **执行推演**：
   - 启动多智能体模拟
   - 实时监控推演过程
   - 查看推演结果和事件发展

4. **世界线管理**：
   - 保存推演项目
   - 增加新的变数和设定
   - 回溯时间线重新推演

5. **结果分析**：
   - 生成推演报告
   - 可视化世界变化
   - 与推演中的人物交互

## 快速开始

### 环境要求

- Node.js 18+
- Python 3.11+
- uv (Python包管理器)

### 安装依赖

```bash
# 安装所有依赖
npm run setup:all
```

### 配置环境变量

```bash
# 复制并编辑环境变量文件
cp .env.example .env
```

### 启动服务

```bash
# 启动前后端服务
npm run dev
```

### 访问地址

- 前端：http://localhost:3000
- 后端 API：http://localhost:5001

## 核心特性

### 1. 世界观构建系统

- **多维度数据管理**：支持管理地理、政治、经济、文化等多个维度的世界观数据
- **AI 辅助提取**：上传文本文件，AI 自动提取关键信息并结构化
- **实体关系管理**：可视化管理人物、国家、组织等实体及其关系

### 2. 双模式推演引擎

- **未知推演**：基于现有设定，预测未来发展
- **已知推演**：细化已有时间线，丰富细节
- **智能体模拟**：多智能体交互，模拟社会演化

### 3. 世界线管理

- **项目持久化**：保存完整的推演项目
- **时间线回溯**：支持回到任意时间点重新推演
- **变数注入**：随时添加新的设定和变量

### 4. 可视化系统

- **世界地图**：交互式地图编辑和展示
- **时间线**：可视化时间线编辑
- **关系图谱**：实体关系网络展示
